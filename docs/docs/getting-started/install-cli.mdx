:::note
This getting started guide is a quick start with [Cloud-based Managed Tracetest](https://app.tracetest.io/). To use [Enterprise On-prem Tracetest, refer to this guide](/install/on-prem-helm).
:::

1. Install the Tracetest CLI.

    <GtagInstallCliTabs></GtagInstallCliTabs>

2. Configure the CLI with your Organization API Key/Token and Environment ID.

    ```bash title="Terminal"
    tracetest configure -t <TRACETEST_API_KEY> --environment <TRACETEST_ENVIRONMENT_ID>
    ```
